Loops in Ruby...

March 30, 2014

If you've experienced pair-programming in Ruby before, it's likely that you've noticed that people have different preferences in loops. Some prefer using each loops and others for loops.
Do you know how many kinds of loops Ruby has? If not, let's take a look at Ruby's loop world together.

  • WHILE loop

    • If you use a while loop, the code inside the loop will repeat itself while the given condition is met.

    while condition{
    code
    }

    while condition do
    code
    end

    For example, if you want to count number of elements in an array using a while loop, your code will look like this.

    i == 0
    count == 0
    while i < array.length{
    count += 1
    i += 1
    }
    return count

  • UNTIL loop

    • An until loop is pretty much similar to a while loop. The only difference is that the code will repeat itself until some condition is met.

    until condition{
    code
    }

    until condition do
    code
    end

    For the same example, you will code like below if you use an until loop.

    i == 0
    count == 0
    until i >= array.length{ count += 1
    i += 1
    }
    return count

  • FOR loop

    • A for loop looks a bit different than the previous two. It takes a variable and states a condition for that variable, only in which the code will be repeated.

    for variable in expression{
    code
    }

    for variable in expression do
    code
    end

    You can solve the same problem using a for loop like this.

    i == 0
    count == 0
    for i in 0..array.length-1{
    count += 1
    i += 1
    }

  • EACH loop

    • What an each loop does is execute the code for each element of whatever your input. This is effective especially you're working on an array or a hash.

    object.each{
    code
    }

    object.each do
    code
    end

    You can code with an each loop for the same problem like this.

    count == 0
    array.each{
    count += 1
    }
    return count

  • TIMES loop

    • In a times loop, you can say how many times you want to repeat the code.

    number.times{
    code
    }

    number.times do
    code
    end

    The solution for the same problem using a times loop will look like this.

    count == 0
    array.length.times{
    count += 1 }
    return count

This is a very introductory explanation of loops in Ruby, but I hope you get the concept! Let's actually use those loops and get familiar with them!
